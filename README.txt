## 一、目录结构
- /css：样式表
- /js：JavaScript文件
- /fonts：字体文件
- /src：源码，html文件之后将转为php文件
- /demand：客户需求、session存储对象源码、参考资料


## 二、部署注意事项
1. 数据库：每一个页面的首部，进行数据库的连接，测试时调用了我方服务器的数据库，拼接时请调用你方的数据库。(数据库建表可以参考文件中的GreenCity.sql文件，文件可以用记事本打开)
2. cookie：进行数据库操作时，利用cookie暂时存储数据，有可能存在cookie命名与你方命名重复的问题。
3. 网页入口：你方登录后的界面应当跳转到 src/List.php 文件。同时，适当地传递session，包括用户信息等。
4. 跳转：之前测试时是用目录下的admin1、admin2、user1、user2，可以无视。
